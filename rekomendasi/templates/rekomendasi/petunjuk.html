Petunjuk Penggunaan

Mahasiswa

  1. Lakukan login/register terlebih dahulu.
  2. Pergi ke halaman profile, kemudian isi peminatan & nilai pada kelas-kelas yang tersedia.
  3. Kembali ke halaman beranda (pencet REKDOS pada navigasi), kemudian pilih atau buat run yang ingin dilakukan.
  4. Partisipasi dalam run dengan menekan tombol 'Bergabung dalam run'
  5. Run hanya dapat dijalankan oleh pembuat run.
  6. Hasil run dapat dilihat pada halaman run.

Dosen

  1. Lakukan login/register terlebih dahulu.
  2. Pergi ke halaman profile, kemudian isi peminatan & kelasyang diajar dari kelas-kelas yang tersedia.
  3. Kembali ke halaman beranda (pencet REKDOS pada navigasi), kemudian pilih atau buat run yang ingin dilakukan.
  4. Partisipasi dalam run dengan menekan tombol 'Bergabung dalam run'
  5. Run hanya dapat dijalankan oleh pembuat run.
  6. Hasil run dapat dilihat pada halaman run.
